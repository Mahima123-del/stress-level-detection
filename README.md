A full-stack group project that detects user stress levels using two modules: camera-based and text-based analysis.  

Key features:  
- User must log in to access the system  
- Home page displays a welcome message with the user's name  
- Navigation bar includes: Home, CameraStress, TextDetection Result, and Profile  
- **CameraStress Module**: Uses computer vision and facial expression analysis to detect stress  
- **TextDetection Module**: Analyzes user-entered text using NLP and machine learning to detect stress level  
- **Profile Page**: Displays photo, User ID, Username, and Email of the user  

Technologies Used:  
- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js or Flask (depending on final version)  
- ML/NLP: Python, OpenCV, Scikit-learn  
- Database: MySQL
